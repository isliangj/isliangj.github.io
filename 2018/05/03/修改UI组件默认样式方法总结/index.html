element 官网方法

请参考 element-自定义主题

修改非通用样式

  • 业务组件的样式覆盖可以考虑在vue单文件组件中覆盖
  • 此时涉及到选择器 优先级 的问题，部分样式需要使用style的 全局样式（即不加scoped）
  • 使用全局样式的时候尽量添加业务组件的根组件选择器，以免影响其他组件中对element组件的样式。
