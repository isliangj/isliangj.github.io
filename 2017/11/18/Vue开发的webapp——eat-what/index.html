2017年11月18日 | 项目总结

项目简介

本项目能解决  你人生的三件大事 —— 早上  吃什么 中午吃什么 晚上吃什么，让有选择  困难症的你不再纠结。

  1. 主页面

通过类似老虎机滚动的形式进行随机选择

  1. 添加美食

会在当前  分类项中添加美食

  1. 切换分页

用滑出主菜单的形式切换  分页，共有两个  页面，‘ 选择 ’ 和 ‘ 列表 ’

  1. 编辑美食

 在 ‘ 列表 ’ 页面中通过  左滑的形式，对美食进行修改  和删除的操作

  • Vue.js 2.0

  • Mint UI

  • Vuex

  • vue-router

关键部分

循环滚动

 与一般 ‘ 老虎机抽奖 ’ 的图片滚动  不同， 这里需要滚动的列表，而且是需要动态添加的， 同时需要做到无缝循环滚动，我们  使
用定时器， 每隔一定时间改变列表的 ‘top’， 在用 ‘transition’ 添加动画效果，为了做到循环滚动， 在滚动的间隙，将  第一项删
除，并 push 到最后一项。

本地存储

 项目 1.0 版的是个人版，所以需要将数据存在本地，只要不清空手机浏览器的缓存，数据一直都在，方便下次使用。所以我们使用了
js 原生的 localStorage，在 Vuex 中，当数据发生改变就将新数据储存一遍，当项目刚启东是， 先调用本地的数据。

另外 localStorage 只支持字符串的形式，所以要通 JSON.stringify 和 JSON.parse 对数据进行转化

移动端 UI 组件 

本项目的 UI 组件使用的是 Mint UI，该组件是 ‘ 饿了么 ’ 开源的移动端组件库，UI 风格和本项目十分吻合，所以使用该组件进行开
发

具体使用参考  Mint UI 官网
